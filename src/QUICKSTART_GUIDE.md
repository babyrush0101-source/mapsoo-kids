# Baseul Kids - 博客和社区系统快速入门指南

## 🎯 快速开始

### 1. 查看博客内容

**路径**: 点击导航栏 "Blog" 或访问 `/blog`

**你会看到**:
- Bento 风格的博客网格布局
- 6 篇高质量初始文章
- 不同大小的卡片（Featured 文章更大）
- 每篇文章显示：标题、摘要、阅读时间、发布日期

**操作**:
- 点击任意文章卡片查看详情
- 观察不同内容类型的渲染效果

### 2. 浏览社区

**路径**: 点击导航栏 "Community" 或访问 `/community`

**你会看到**:
- Bento 风格的社区帖子网格
- 6 篇初始社区帖子
- 热门帖子带有 "Hot" 标签
- Featured 帖子显示更大

**操作**:
- 未登录：可浏览所有帖子
- 登录后：点击 "Create Post" 发布新帖子
- 点击帖子查看详情和回复

### 3. 管理员功能

#### 登录为管理员

1. 访问 Admin Login (通常在 footer 或直接访问 `/admin-login`)
2. 使用凭证:
   ```
   Email: admin@baseul.com
   Password: admin123
   ```
3. 进入 Admin Dashboard

#### 管理博客文章

**从 Admin Dashboard**:
1. 点击 "Manage Blog Posts"
2. 你会看到所有博客文章列表（包括草稿）

**创建新文章**:

1. 点击 "New Blog Post"
2. 填写基本信息:
   - **标题** (必填)
   - **摘要** (必填) - 显示在博客列表
3. 选择内容类型:
   - **Text Only** - 纯文字文章
   - **Image + Text** - 图文混合
   - **Video + Text** - 视频 + 文字
4. 添加内容块:
   - 点击 "Add Text" 添加文本块
   - 点击 "Add Image" 添加图片块
   - 点击 "Add Video" 添加视频块
5. 编辑每个块:
   - **文本块**: 输入文章内容
   - **图片块**: 
     - URL: 图片链接 (使用 Unsplash)
     - Caption: 图片说明 (可选)
   - **视频块**:
     - URL: YouTube 嵌入链接 (格式: `https://www.youtube.com/embed/VIDEO_ID`)
     - Caption: 视频说明 (可选)
6. 设置选项:
   - ☑️ **Publish immediately** - 立即发布（否则保存为草稿）
   - ☑️ **Feature this post** - 在博客列表中显示更大
7. 点击 "Create Post"

**编辑现有文章**:
1. 在博客列表中点击 "Edit" 图标 (铅笔)
2. 修改内容
3. 点击 "Update Post"

**其他操作**:
- **👁️ / 👁️‍🗨️ 图标**: 发布/取消发布
- **🗑️ 图标**: 删除文章
- **查看状态**:
  - 🟢 Published - 已发布
  - ⚪ Draft - 草稿
  - 🟣 Featured - 精选

## 📝 内容类型详解

### 纯文字文章 (Text Only)

**适用场景**:
- 深度分析文章
- 列表式内容
- 教程和指南

**示例**: "10 Tips for Supporting Your Toddler's Language Development"

**结构**:
```
Block 1: 引言段落
Block 2: 主要内容
Block 3: 总结
```

### 图文文章 (Image + Text)

**适用场景**:
- 视觉化教程
- 案例研究
- 环境/空间展示

**示例**: "Creating a Montessori-Inspired Learning Space"

**结构**:
```
Block 1: 引言文字
Block 2: 图片 + 说明
Block 3: 详细解释
Block 4: 图片 + 说明
Block 5: 总结
```

### 视频+文字文章 (Video + Text)

**适用场景**:
- 产品介绍
- 演示教程
- 活动回顾

**示例**: "Introducing Baseul Memory"

**结构**:
```
Block 1: 产品介绍文字
Block 2: 视频演示
Block 3: 功能详解
Block 4: 总结和行动号召
```

## 🖼️ 获取图片 URL

### 推荐来源: Unsplash

1. 访问 [Unsplash.com](https://unsplash.com)
2. 搜索相关主题（如 "children learning", "education", "parenting"）
3. 点击喜欢的图片
4. 右键点击图片 > "复制图片地址"
5. 粘贴到博客编辑器的图片 URL 字段

**提示**: Unsplash 图片已经过本系统验证，加载速度快且质量高。

### 示例 Unsplash 搜索词
- "children learning"
- "toddler education"
- "parent child bonding"
- "montessori classroom"
- "kids activities"
- "child development"

## 🎥 添加 YouTube 视频

### 获取嵌入链接

1. 打开 YouTube 视频
2. 点击 "分享" → "嵌入"
3. 复制 `<iframe>` 标签中的 `src` 地址
4. 格式应该是: `https://www.youtube.com/embed/VIDEO_ID`
5. 粘贴到视频 URL 字段

**示例**:
```
https://www.youtube.com/embed/dQw4w9WgXcQ
```

## 🎨 Bento 布局说明

### 什么是 Bento 布局?

Bento（盒饭）布局是一种现代的网格设计风格，特点是:
- 不规则的网格大小
- 视觉层次分明
- 留白和间距舒适
- 动态且吸引人

### 如何利用 Bento 布局?

**Featured 文章**:
- 勾选 "Feature this post" 使文章在列表中占据更大空间
- 适合重要的公告、深度文章、产品介绍

**普通文章**:
- 不勾选 Featured，文章将以标准大小显示
- 适合常规更新、技巧分享

**系统自动调整**:
- 布局会自动组合不同大小的卡片
- 移动端自动调整为单列
- 平板和桌面端显示多列网格

## 📱 响应式体验

### 桌面端 (>1024px)
- 博客: 4 列网格
- 社区: 3 列网格
- Featured 内容占 2x2 或 2x1 空间

### 平板端 (768px-1024px)
- 博客: 2 列网格
- 社区: 2 列网格
- Featured 内容占 2x2 空间

### 移动端 (<768px)
- 博客: 1 列
- 社区: 1 列
- 所有卡片全宽显示

## 🔧 故障排除

### 图片不显示

**检查**:
1. URL 是否正确
2. 图片是否可公开访问
3. URL 是否以 `https://` 开头

**解决**:
- 使用 Unsplash 图片（推荐）
- 检查图片链接是否失效
- 系统会自动显示占位符渐变背景

### 视频无法播放

**检查**:
1. URL 格式是否正确
2. 是否使用了嵌入链接（不是观看链接）
3. 视频是否设置为私密

**解决**:
- 使用 YouTube 嵌入链接格式
- 确保视频可公开访问
- 检查浏览器是否阻止了 iframe

### 博客不显示

**检查**:
1. 文章是否已发布（published = true）
2. 是否以管理员身份登录（可以看到草稿）

**解决**:
- 在管理员界面检查文章状态
- 点击眼睛图标发布文章

## 💡 最佳实践

### 博客创作

1. **标题**: 简洁有力，50 字符以内
2. **摘要**: 1-2 句话总结文章，吸引读者点击
3. **内容块**: 
   - 文本块不要太长（200-400 字一段）
   - 图片块与文字交替，增加可读性
   - 视频放在文章开头或关键位置
4. **Featured**: 只标记真正重要的文章（建议 <30%）

### 社区管理

1. **鼓励参与**: 定期发布话题帖
2. **及时回复**: 提高社区活跃度
3. **Featured 帖**: 标记优质讨论

### 图片选择

1. **相关性**: 图片要与内容主题匹配
2. **质量**: 使用高分辨率图片
3. **一致性**: 保持整体风格统一
4. **版权**: 使用 Unsplash 等免费图库

## 🚀 高级技巧

### 创建系列文章

1. 在标题中使用编号（如 "Part 1", "Part 2"）
2. 在每篇文章末尾添加系列导航
3. 使用一致的封面图风格

### 优化阅读体验

1. 使用小标题分段（在文本块中使用 `**加粗**`）
2. 列表使用 `•` 或 `-` 开头
3. 重要信息使用 `**加粗**` 强调
4. 图片说明简洁明了

### 社区互动

1. 在博客末尾引导读者到社区讨论
2. 社区热门话题可发展为博客文章
3. 建立内容循环：博客 → 社区 → 博客
