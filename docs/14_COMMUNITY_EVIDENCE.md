# Community evidence ledger

This ledger records only public, independently reproducible use of Mapsoo Worldsmith and the maintainer response that followed. It exists to prevent maintainer tests, automated downloads, stars, plans, or private claims from being presented as external adoption.

## Current state

As of 2026-07-19, **no independent user report has been recorded**. The public [first-import index](https://github.com/babyrush0101-source/mapsoo-kids/issues/12) is ready, but an empty feedback issue is an invitation, not adoption evidence. STOYO remains a planned consumer scenario and is not counted here.

## Evidence requirements

A row may be added only when all required evidence is public-safe and linked:

1. a report created by someone other than the Primary Maintainer;
2. the exact Mapsoo release tag, pack filename and SHA-256;
3. operating system and exact Godot version;
4. a reproducible success or failure result with private data removed;
5. the maintainer's public triage or response; and
6. any resulting documentation, pull request or release link.

Anonymous download counts, CI jobs, bot activity, maintainer-run smoke tests, unverifiable private messages and requested testimonials do not qualify.
