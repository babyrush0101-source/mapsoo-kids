# v0.1.0-alpha.6 candidate release visuals

`renderer-v0.1.0-alpha.6.html` is an intentionally minimal, local-only source file. The Alpha.6 release registry declares no renderer frames and no image outputs because itch.io login, upload, and visual production remain postponed.

No Alpha.5 image is renamed or reused as Alpha.6 evidence. If visual production resumes, new images must be rendered from the exact Alpha.6 fixture, registered under a versioned media directory, inspected, and verified before any itch.io preview or upload. Until then, this file documents the deferral and makes no claim that an itch.io page, Godot Asset Library entry, or external adoption exists.
