# v0.1.0-alpha.1 release visuals

`renderer.html` is the deterministic source for the itch.io cover and five release screenshots. It uses only:

- the real Sunny Meadow pack preview and atlases;
- the real browser workbench screenshot;
- verified manifest metadata and Godot CLI results;
- local CSS and system fonts.

It does not call an image-generation model or load remote assets. This preserves the release disclosure that the distributed artwork is generated by deterministic, self-contained procedural code.

| Query | Output size | Purpose |
| --- | ---: | --- |
| `?frame=cover` | 1260×1000 | itch.io cover at the 315:250 ratio |
| `?frame=hero` | 1600×900 | actual generated pack preview |
| `?frame=workbench` | 1600×900 | World Spec and validation workbench |
| `?frame=contents` | 1600×900 | exact pack file and atlas contents |
| `?frame=godot` | 1600×900 | real Godot 4.3/4.7 CLI import evidence |
| `?frame=contract` | 1600×900 | open asset-contract flow |

PNG outputs belong in `docs/media/v0.1.0-alpha.1/itch/`. Verify dimensions and inspect every image before upload. The Godot slide intentionally presents CLI evidence rather than pretending to be an editor screenshot; replace it only with a real editor capture from the exact attached pack.
