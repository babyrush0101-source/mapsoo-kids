# Mapsoo Worldsmith v0.1.0-alpha.4

Alpha.4 is the published Playable Terrain Pack prerelease. It advances the portable PNG + JSON contract from a flat sample into explicit Ground, Water, Roads, and Props layers that the separately installed Godot importer can turn into editable terrain resources.

## Highlights

- Published export: `mapsoo-sunny-meadow-v0.1.0-alpha.4.zip`.
- Pinned Sunny Meadow SHA-256: `a57e810baaf2f015d7db96bf0e88ab7b6340d476a61ade7447735a6109b8fb35`.
- Pack schema `0.2.0` with exactly four declared layers: Ground, Water, Roads, and Props.
- One 256 × 192 terrain atlas declares 35 stable terrain tiles: 3 ground tiles, 16 cardinal Water masks, and 16 cardinal Road masks.
- One 192 × 32 prop atlas declares 6 sprites: tree, rock, flower, shrub, log, and marker.
- Water and Roads have separate Godot `MATCH_SIDES` terrain sets. Water tiles use full-cell blocking collision on the `world-blocking` physics layer; Ground and Roads have no collision.
- The deterministic 24 × 16 Sunny Meadow example contains 384 ground cells and 29 placed prop instances.
- The exact 12-file ZIP contract remains one manifest plus 11 payload records, including generation receipt `0.2.0`.
- The asset ZIP remains executable-free and contains only PNG, JSON, and Markdown.

## Compatibility and provenance

The schema 0.2 pack requires Mapsoo Importer `0.1.0-alpha.4` and declares Godot 4.3 as its minimum. The release gates are scoped to Godot 4.3 and Godot 4.7 on Linux and Windows. Older schema 0.1 fixtures and published hashes remain immutable.

Generation receipt `0.2.0` binds the shipped World Spec and seed to the built-in `procedural-terrain-v2@0.2.0` provider, local execution, versioned transformations, the CC0 notice, and `contains_generative_ai: false`. Procedural graphics and generated map assets are CC0-1.0. Bundled schemas, documentation, and importer code retain MIT notices. Codex assisted project code and text, not the pixel-art output.

## Scope boundary

Alpha.4 provides deterministic layered terrain, editable `MATCH_SIDES` metadata, explicit scene tile IDs, six prop sprites, and basic Water blocking. It is not a complete game, a navigation system, a pathfinding solution, or a claim that every generated world is production-ready. Road gaps through Water are explicit ford/gap cells; this release does not infer bridges.

## Release state

The [tag workflow](https://github.com/babyrush0101-source/mapsoo-kids/actions/runs/29680728876) rebuilt and verified the fixed pack, audited the exact 11-attachment set, and passed the Linux/Windows Godot 4.3/4.7 matrix before publication. A separate ledger change pins every public attachment digest and advances the public download links to Alpha.4.

itch.io upload is intentionally postponed. The versioned page copy and renderer are offline operator inputs for later manual review; this release does not claim that an itch.io page exists.
