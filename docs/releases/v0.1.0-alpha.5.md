# Mapsoo Worldsmith v0.1.0-alpha.5

> Published as a GitHub prerelease on 2026-07-19. The 12 public attachment digests are pinned in the release registry; itch.io publication remains postponed.

Alpha.5 adds deterministic semantic places to the existing layered terrain pack. It is still an early, engine-neutral asset contract—not a complete game, quest system, navigation system, or claim of production readiness.

## Highlights

- World Spec `0.2.0` optionally declares 1–8 stable place IDs.
- Six public place kinds: spawn, settlement, landmark, resource, encounter, and exit.
- Four deterministic placement rules: center, near-water, on-road, and map-edge.
- Pack schema `0.3.0` binds `runtime/places.json`, its `0.1.0` schema, and `atlases/places.png` by path and SHA-256.
- The browser preview shows the resolved markers and an accessible place list from the same records.
- The Godot importer derives a `Places` node with queryable `Marker2D` anchors while preserving the existing created / unchanged / updated / conflict ownership boundary.
- Alpha.1–Alpha.4 release files, schemas, tags, and digests remain unchanged.

## Verified example

The executable-free Sunny Meadow example contains 15 files: one manifest and 14 payload records. Its canonical ZIP SHA-256 is:

```text
8d86124a4a37fa4a78487c4e91cb7f5024561f140814a5fd139c5b93fde54f36
```

The pack contains only PNG, JSON, and Markdown. Install the importer separately from this repository. Hashes prove byte consistency, not publisher identity.

## Compatibility and limits

- Targeted import matrix: Godot 4.3 and Godot 4.7 on Linux and Windows.
- Portable PNG and JSON remain the source of truth; generated Godot resources are managed derivatives.
- Stable place IDs are the compatibility surface. Coordinates are deterministic for this pack version but may change in a later world/pack version.
- No navigation mesh, pathfinding, bridge inference, quests, dialogue, NPC logic, save system, or runtime gameplay is included.
- STOYO is a planned consumer scenario, not evidence of production adoption.
- No independent user report has been recorded yet.
- itch.io upload remains postponed; no public itch.io page is claimed.
