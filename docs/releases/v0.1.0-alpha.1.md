# Mapsoo Worldsmith v0.1.0-alpha.1

## 中文

这是 Mapsoo Worldsmith 的第一个公开预发布版本：把版本化 World Spec 转换为可重复生成、可校验、适合 Godot 4.3+ 的 2D 像素世界素材包。

### 本版本包含

- meadow、desert、snowfield 三种离线程序化世界配方；
- 相同 World Spec 与 seed 得到相同地图结果；
- 浏览器内预览和导出前校验；
- 不含可执行代码的 PNG + JSON Sunny Meadow 示例包；
- manifest、逐文件 SHA-256、generation receipt 与 CC0-1.0 素材许可；
- 单独安装的 MIT Godot importer，可派生 `TileSet`、`TileMapLayer` scene 与 prop sprites；
- Godot 4.3 与 4.7 smoke-test 契约。
- 75 秒中英双语、无音轨的 H.264 候选包证据视频。

### 快速开始

1. 下载并解压 `mapsoo-sunny-meadow-v0.1.0-alpha.1.zip`；
2. 仅从本官方 release 下载 `mapsoo-godot-importer-v0.1.0-alpha.1.zip`，将其中的 `addons/mapsoo_importer/` 复制到 Godot 4.3+ 项目；
3. 在 Godot 中启用 **Mapsoo Pack Importer**；
4. 选择素材包根目录中的 `mapsoo.manifest.json`；
5. 打开生成到 `res://mapsoo_imports/<pack-id>/` 的 world scene。

### 安全与许可

素材 ZIP 只允许 PNG、JSON 和 Markdown，不携带 GDScript 或 addon。请勿启用第三方素材包附带的脚本。包内 SHA-256 证明文件与 manifest 自洽，但不代替发布者身份验证。源码与 importer 使用 MIT；本 release 的程序化示例素材使用 CC0-1.0，具体以各压缩包内许可为准。

### 已知限制

- 当前 alpha 使用自包含程序化生成器，不包含图像模型生成内容；
- 每个世界当前只有一个 biome，地图为正交俯视像素风；
- importer 需要先解压素材 ZIP，不直接打开不可信 ZIP；
- 重新导入与冲突处理策略仍会继续完善；
- v1.0 前格式可能发生破坏性变化。

## English

This is the first public pre-release of Mapsoo Worldsmith. It turns a versioned World Spec into a deterministic, validated 2D pixel-world asset pack for Godot 4.3+.

### Included

- Offline meadow, desert, and snowfield recipes;
- deterministic map output for the same World Spec and seed;
- in-browser preview and pre-export validation;
- an executable-free PNG + JSON Sunny Meadow example pack;
- a manifest, per-file SHA-256 records, generation receipt, and CC0-1.0 asset license;
- a separately installed MIT Godot importer that derives a `TileSet`, `TileMapLayer` scene, and prop sprites;
- a Godot 4.3 and 4.7 smoke-test contract.
- a silent, bilingual 75-second H.264 release-candidate evidence video.

### Quick start

1. Download and extract `mapsoo-sunny-meadow-v0.1.0-alpha.1.zip`.
2. Download `mapsoo-godot-importer-v0.1.0-alpha.1.zip` only from this official release and copy `addons/mapsoo_importer/` into a Godot 4.3+ project.
3. Enable **Mapsoo Pack Importer** in Godot.
4. Select `mapsoo.manifest.json` from the extracted pack root.
5. Open the generated world scene under `res://mapsoo_imports/<pack-id>/`.

### Security and licensing

The asset ZIP permits only PNG, JSON, and Markdown data files; it contains no GDScript or addon. Never enable scripts bundled by a third-party asset pack. SHA-256 values prove internal consistency, not publisher identity. Source code and the importer are MIT-licensed; the procedural example assets are CC0-1.0. The licenses inside each archive are authoritative.

### Known limitations

- This alpha uses a self-contained procedural generator and contains no image-model output.
- Each world currently uses one biome and an orthogonal top-down pixel-art layout.
- The importer operates on an extracted pack and does not directly open untrusted ZIP files.
- Re-import and conflict handling will continue to evolve.
- Breaking format changes remain possible before v1.0.
