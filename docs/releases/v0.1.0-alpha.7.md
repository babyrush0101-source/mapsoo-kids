# Mapsoo Worldsmith v0.1.0-alpha.7

Alpha.7 turns the three-world Workbench gallery into three independently downloadable, deterministic Godot asset packs.

## Included worlds

| World | Biome | Pack SHA-256 |
| --- | --- | --- |
| Sunny Meadow | meadow | `6113b30fec3615b72730d8d775919aa3c5552285c614b6916a109b887ab8012c` |
| Dustwind Outpost | desert | `d6dd38a47522f45d24184d9b6869d92b89cc2ae3ad1c2ca1eab0b9cf4b13a502` |
| Frostwatch Vale | snow | `35a49edd901becae1422731a132803eebaf07659fc3d69efa7d39cd1e87b9e12` |

Each executable-free ZIP contains one versioned root, 17 manifest-covered PNG/JSON/Markdown payloads, a generation receipt, semantic places, place-linked structures, schemas, licensing, and per-file SHA-256 records.

## Contract changes

- Pack Schema advances to `0.5.0` and binds the Alpha.7 exporter/importer version.
- Resolved places advance to `0.3.0`; resolved structures advance to `0.2.0`.
- The Godot importer adds Pack Schema `0.5.0` while remaining compatible with Pack Schemas `0.1.0` through `0.4.0`.
- The release registry supports multiple official packs while preserving Alpha.1–Alpha.6 bytes and public digests.

## Verification

- One real Chrome run exports all three ZIPs; a second run must reproduce the pinned bytes.
- Linux and Windows on Godot 4.3 and 4.7 each import all three exact packs.
- Every exact pack proves `created → unchanged`, then a managed-scene edit proves `conflict` with modified bytes preserved.
- Release construction, checksums, manifests, receipt policies, and ZIPs are reproducible.
