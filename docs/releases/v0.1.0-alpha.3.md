# Mapsoo Worldsmith v0.1.0-alpha.3

Alpha.3 makes Godot re-import safe and explicit while preserving the portable PNG + JSON pack contract and both immutable historical releases.

## Highlights

- Current workbench export: `mapsoo-sunny-meadow-v0.1.0-alpha.3.zip`.
- Pinned Sunny Meadow SHA-256: `af95a4e57187fb85d06e34ccb0e1a1b1dba9b91e8989debf4c30a93108589696`.
- Exact 12-file ZIP: one manifest and 11 payload records, including generation receipt 0.2.0.
- Godot importer `0.1.0-alpha.3` records generated ownership in `mapsoo.import-state.json`.
- Re-import outcomes are `created`, `unchanged`, `updated`, or fail-closed `conflict`.
- Same-parent staging, source snapshots, final-baseline checks, backup checks, deterministic promotion rollback, and race tests protect the process-level transaction.
- Linux and Windows release gates are configured to run the exact candidate pack on Godot 4.3 and 4.7.
- The asset ZIP remains executable-free and contains only PNG, JSON, and Markdown.

## Reliability boundary

The importer preserves a clean previous output when validation, staging, or deterministic promotion fails. This is process-level rollback, not a claim of crash- or power-loss atomicity. Concurrent writers are outside the alpha.3 contract. Legacy output without ownership state is not silently adopted.

## Compatibility and provenance

The data format remains compatible with importer alpha.1 for first import; importer alpha.3 is recommended and distributed for the safe re-import contract. Procedural graphics and generated map assets are CC0-1.0. Bundled schemas, documentation, and importer code retain MIT notices. `contains_generative_ai: false` describes the pack graphics; Codex assisted code and text, not the pixel-art output.

## Release process

The tag workflow builds a draft prerelease, checks the exact attachment set and fixed pack hash, and requires Godot 4.3/4.7 on Linux and Windows before publication. Public web download links remain on alpha.2 until the alpha.3 prerelease is published and a separate ledger change pins every public attachment digest.

itch.io upload is intentionally postponed. The verified offline operator kit remains available for later manual review; this release does not claim an itch.io page exists.
