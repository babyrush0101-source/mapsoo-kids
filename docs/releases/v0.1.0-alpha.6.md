# Mapsoo Worldsmith v0.1.0-alpha.6

> Published GitHub prerelease at tag commit `315bc1b`. The release workflow succeeded with 13 audited attachments. itch.io publication remains postponed.

Alpha.6 extends the semantic-place contract with deterministic, place-linked exterior structure artwork. It remains an early, engine-neutral asset contract—not a complete game, building-interior system, navigation system, or production-readiness claim.

## Highlights

- World Spec `0.3.0` can declare optional structures linked to existing stable place IDs.
- Four bounded exterior archetypes: cottage, workshop, tower, and shrine.
- Pack schema `0.4.0` binds `runtime/structures.json`, `atlases/structures.png`, the World Spec, and `runtime/places.json` by path and SHA-256.
- The browser preview draws structures from the same deterministic source used for the exported transparent atlas, with an independent layer toggle and readable list.
- The Godot importer derives a managed `Structures` container with `Sprite2D` children and queryable place/structure metadata while retaining the created / unchanged / updated / conflict ownership boundary.
- Alpha.1–Alpha.5 release files, schemas, tags, and public digests remain unchanged.

## Compatibility and limits

- The successful release workflow verified Godot 4.3 and Godot 4.7 on Linux and Windows.
- Portable PNG and JSON remain the source of truth; generated Godot resources are managed derivatives.
- Stable structure IDs and `place_id` links are the Alpha.6 compatibility surface. Coordinates, atlas regions, and art may change only through a future explicit world/pack version.
- Structures are exterior artwork only. No interiors, doors, interaction, collision, navigation, quests, dialogue, NPC logic, economy, or save system is included.
- STOYO is a planned consumer scenario, not evidence of production adoption.
- No independent user report or external adoption is claimed.
- itch.io upload remains postponed; the repository contains only an offline Draft source and does not claim a public itch.io page.
