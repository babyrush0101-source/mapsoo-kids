## Godot quick start

1. Install **Mapsoo Pack Importer** from the official repository: https://github.com/babyrush0101-source/mapsoo-kids
2. Extract this data pack.
3. In Godot choose **Project → Tools → Import Mapsoo Pack...** and select the extracted `mapsoo.manifest.json`.
4. Open the generated scene under `res://mapsoo_imports/sunny-meadow/`.

The imported TileSet includes separate match-sides terrain sets for Water and Roads. Water tiles have full-cell blocking collision on the `world-blocking` physics layer. The demo scene uses explicit tile IDs, so its first import is deterministic.

## Security boundary

This asset pack intentionally contains no executable GDScript or addon. Do not enable scripts copied from a third-party asset pack. SHA-256 records verify pack consistency, not publisher identity.

PNG and JSON remain the portable source of truth; the separately installed importer derives Godot resources inside the editor.
